# 大语言模型编程马拉松：Langchain 全景实战训练营

### 🎯 课程目标

- **快速入门**：在 2 天课程内，亲手运行和搭建自己的 🤖 LLM 应用。
- **确保收获**：每位学员都不会空手而归。
- **全产业链**：覆盖产品/编码/发布/运维/模型部署的完整链条。

### 👥 课程受众

- 希望成为 👥 LLM 产品经理 的专业人士，希望了解 LLM 的商业和技术应用。
- 想要转型成为 👨‍💻 LLM 软件工程师 的开发者，希望通过实际编程加深对 LLM 的理解。

**⭕️ 入门门槛**：开箱即用的积木代码 + ChatGpt + 组内助理，点击即可运行。

### 💰 课程收益

- **跨越门槛**：快速消除初入人工智能领域的陌生与畏惧感。
- **认知提升**：深入理解大语言模型的工作原理及其在多种场景下的实际应用。
- **编程技能**：非编程背景的学员也能通过课程提供的积木代码尝试编写和改进代码。
- **真实体验**：通过修改代码参数、问题内容来感知 LLM 的能力与局限。
- **自学赋能**：通过对 Langchain 框架的解读，借助于人工智能辅助编程，让没有 AI 和 python 基础的学员在课后仍然能编写和扩展应用。

### 📖 课程大纲

#### 📚 概述：LLM 简介与核心概念

- 什么是大语言模型 LLM
- LLM 的发展历史
- LLM 是如何工作的
- LLM 与 AI 的关系

#### 💻 开发篇

##### 🏁 第一课：Langchain 智能体概览

- 智能体创建及其主要参数。
- 组合各类工具与模型。
- 调用外部 API、知识库和搜索引擎，自动完成复杂任务。
- 内置记忆、规划与错误处理机制，实现更智能的交互体验。

##### 🏁 第二课：Hello World!

- 亲手操作 LLM，从最基础的“Hello World!”程序开始，逐步学习如何调用 LLM 进行简单的文本生成。
- 解释 API 的使用方法与调用参数。
- 讲解如何安全有效地管理 API 账号与密钥。
- 理解 LLM Api 的计费方式与持续性成本。

##### 📝 第三课：提示词工程

- 学习如何设计有效的提示词以引导 LLM 生成更高质量的内容。
- 介绍不同类型的提示词模板，如静态提示、动态提示。
- 探讨如何利用用户输入调整提示词以改善交互质量。

##### 📚 第四课：检索增强生成（RAG）

- 介绍如何通过集成外部数据来提高 LLM 的输出质量和相关性。
- 基于文档的集成方法。

##### 🖼️ 第五课：MultiModal 多模态

- 理解多模态的运行机制。
- 学习基于图片的多模态描述与识别。

##### 🧠 第六课：LangGraph 智能体深度剖析

- 了解智能体与工具的基本运行原理。
- 思考与工具识别。
- 自定义工具。
- 人工介入接口。
- 短期记忆与长期记忆压缩。

##### 🧠 第七课：DeepAgents 深度智能体框架

- 创建能自主思考和智能体。
- 核心机制剖析（如 Todo list 的创建）。
- 内置本地文档读写工具。
- 短期记忆与长期记忆压缩。
- 跨会话、跨项目的知识库。

##### 🏗️ 第七课：大语言模型应用架构

- 了解大语言模型应用如何嵌入到常见的软件中。
- 以 WebApi 运行/接入 Web/接入微信小程序/接入 Vscode。
- 用真实案例了解大语言模型的 DevOps 及运营流程。
  - 某人工智能考试系统的 LLM 后台
  - 某人工智能需求分析系统的 LLM 后台
  - 某保健类小程序的 LLM 后台

#### 🏭 生产运维篇

##### 🧑‍💻 第八课：LangFlow 无代码开发平台

- 如何完全不写代码开发 LLM 应用。
- 基于 LangFlow 的可视化开发。

##### 🖥️ 第八课：Ollama 大模型本地部署

- 了解如何使用 Ollama 本地部署大模型。
- 了解大模型参数代表的含义。

##### 🌐 第九课：OpenWebUI 开源开箱即用平台

- 了解如何在企业内部使用 OpenWebUI + Ollama 启动第一个免开发智能体。
- 如何配置参数实现本地知识库调用。
- 如何让智能体调用外部工具。
- 如何开发和添加自研工具。
- 比较不同模型处理同一问题的能力。

##### 第十课：大语言模型产品运营

- 不同模型的性能、价格比较与选择
- 大语言模型的自动化测试
- 使用 Ragas 为大模型应用评分
- 使用 LangSmith 统计大模型应用的成本、质量、客户反馈
- 合规性检查
  - 使用阿里工具
  - 使用 Langchain 工具
