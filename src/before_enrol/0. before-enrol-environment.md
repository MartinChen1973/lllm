# 学前准备——Tools 基础工具

## GitHub Desktop 安装

**什么是 GitHub**

课程中使用 GitHub 共享教程代码，学员也可在修改代码后上传到自己的 GitHub 仓库中。

**注意** 如果用户已经在使用其他平台如gitlab、gitee也可以。企业内部的平台也可。

**下载安装**

1. 打开 [GitHub Desktop 官方下载页面](https://desktop.github.com/)。
2. 点击 **Download for Windows**（或其他适合您操作系统的版本）。
3. 下载完成后，双击安装文件并按照提示完成安装。
4. 安装完成后，启动 GitHub Desktop。若尚无 GitHub 账号，请先访问 [https://github.com/signup](https://github.com/signup) 注册；也可使用微软账号授权注册。
5. 在 GitHub Desktop 中点击「Sign in to GitHub.com」或「Sign in with your browser」，在浏览器中完成登录。

**完成标准**

- 登录成功后，GitHub Desktop 应显示您的 GitHub 用户名和头像，并能看到仓库列表或欢迎界面。

## 安装 Python（3.12.12 及以上版本）

请确保已安装 Python 3.12.12 或更高版本。

- 官方下载地址：[Python 3.12.12](https://www.python.org/downloads/release/python-31212/)

**重要：安装时必须勾选以下选项**

在安装程序第一个界面（「Install Python」界面）底部，请务必勾选：

- **英文版本**：「Add Python to PATH」或「Add python.exe to PATH」
- **中文版本**：「将 Python 添加到 PATH」或「添加 Python 到环境变量」

该选项将 Python 加入系统环境变量，以便在任意位置通过命令行使用 `python` 命令。

若需自定义安装，可参考以下选项（缺省选中的建议保留）：

- 第一页：选中「Use admin privileges when installing py.exe」，点击「Customize installation」
- Optional Features：选中「py launcher」「for all users」
- Advanced Options：选中「Install … for all users」、**「Add Python to environment variables」**（有时需手工设置环境变量）

**完成标准**

- 打开 PowerShell 或命令提示符（CMD），输入 `python --version`，应显示版本号（如 `Python 3.12.12`，应为 Python 3.12.12 或更高）。
- 若提示「不是内部或外部命令」，说明未正确添加到 PATH，需重新安装并勾选上述「Add Python to PATH」/「将 Python 添加到 PATH」选项。
- 若仍无效，请重启电脑后再试。

**注意**

- 很多AI框架已经不再支持Python 3.8 之前的版本（2024 年 5 月）。

## 安装代码编辑器

请安装以下任意一个代码编辑器。现场实践推荐使用 **Cursor**。

### 使用 VS Code ide

**为何推荐 VS Code**

VS Code 是当前市场份额最大的工具，且免费。拥有众多扩展（Extension），可编辑、预览、查看绝大多数代码与文档类型。

**下载安装**

- 下载地址：[https://code.visualstudio.com/](https://code.visualstudio.com/)（或 [Download](https://code.visualstudio.com/Download)）

**完成标准**

- 运行 VS Code，点击菜单栏「文件」→「打开文件夹」，选择一个包含文档的目录（如「文档」文件夹），确认能在左侧文件列表中看到该目录中的文件。

**VS Code 中需要安装以下插件（Extension）：**

- python （解析和运行 python）
- GitHub Markdown Preview （必装，MD 格式是本课程优先使用的文档存储格式）

可选装如下插件：

- GitHub Copilot（若使用 Copilot，参见下文「准备 AI 辅助编程工具」）

**配置 Python 解释器**

- 打开 VS Code
- 按 `Ctrl + Shift + P` 打开搜索框
- 输入 >Python: Select Interpreter
  - 如果直接在上面的搜索框中输入，需要输入 >Python: Select Interpreter（即需要手工输入一个>符号）
- 此时可能会看到多个安装的 Python 版本，请选择本课程推荐的版本。
  - 错误选择可能会导致某些问题，如提示某个包（如 dotenv）未找到，只需再次选择正确的版本即可。

### 使用 Cursor ide（强烈推荐）

Cursor 是基于 VS Code 的 AI 编程编辑器，是vs code的上位替代品。内置 AI 辅助功能，适合课程实践。

**下载安装**

- 下载地址：[https://cursor.sh/download](https://cursor.sh/download)

**完成标准**

- 运行 Cursor，点击菜单栏「File」→「Open Folder」，选择一个包含文档的目录（如「文档」文件夹），确认能在左侧文件列表中看到该目录中的文件。

**配置 Python 解释器**

- 与 VS Code 相同：按 `Ctrl + Shift + P`，输入 `>Python: Select Interpreter`，选择本课程推荐的 Python 版本。

### 使用 Visual Studio / IntelliJ IDEA 等

正在使用这些平台进行开发的用户，需要自己查证可以使用哪些AI辅助编程工具，以及支持哪些语言。

**AI 辅助编程插件推荐（2026年1月30日）**

- **GitHub Copilot**
  - 支持 Visual Studio、IntelliJ IDEA、PyCharm 等 JetBrains 系列 IDE，可以直接在各自插件市场或 IDE 内部搜索“Copilot”安装插件。

- **TabNine**
  - 支持 JetBrains 系列 IDE 和 Visual Studio，可在对应插件市场或 IDE 内搜索“TabNine”安装。

- **通义灵码（Alibaba Lingma）**
  - 支持 JetBrains 系列 IDE、Visual Studio 等，可在插件市场搜索“通义灵码”或“Lingma”安装；国内用户建议优先体验。

- **腾讯云代码助手 CodeBuddy**
  - 支持 JetBrains 系列 IDE 及 Visual Studio，在插件市场或 IDE 内搜索“CodeBuddy”即可安装，适合国内网络环境。

**安装提示**

- 在 Visual Studio 或 IntelliJ IDEA 主界面，依次点击「Settings」（或「File」→「Settings」），选择「Plugins」，搜索所需插件名，点击「Install」即可。
- 插件安装后按提示登录、注册即可使用 AI 自动补全和辅助编程功能。

**补充说明**

- 若想体验 AI 编程辅助，强烈推荐安装 Copilot、TabNine 等主流插件，并结合自身实际选择国内外生态适用的工具。
- JetBrains 平台插件市场访问速度可能受网络影响，必要时请配置合适的网络环境。
