# 🧠 Cursor：零基础编程入门指南

---

## ✅ 什么是 Cursor？

**Cursor** 是一个智能代码编辑器，它内置了强大的 AI 助手。即使你完全不会编程，也可以告诉 Cursor 你想要什么，它会帮你写出代码。

**简单理解**：就像你告诉一个会编程的朋友"帮我做一个计算器"，然后他帮你写出来一样。Cursor 就是那个会编程的朋友！

---

## 🟢 第一步：安装 Cursor

1. 访问 [https://www.cursor.sh](https://www.cursor.sh)
2. 下载适合你操作系统的版本（Windows/Mac/Linux）。
3. 像安装普通软件一样安装它。
4. 使用 GitHub（推荐）或 Google 账号登录。

---

## 🛠️ 第二步：创建你的第一个项目

### 📁 1. 打开或创建文件夹

- 点击 **"文件"** → **"打开文件夹"**
- 选择一个空文件夹（或者新建一个文件夹，比如叫 `my_first_project`）

### 📄 2. 创建你的第一个文件

- 在左侧文件夹列表中，点击 **"新建文件"** 按钮（或者右键点击文件夹 → 新建文件）
- 给文件起个名字，比如 `calculator.py`（Python 文件）或 `todo.html`（网页文件）

> 💡 **小提示**：文件名后面的 `.py`、`.html` 等是文件类型，告诉 Cursor 这是什么类型的代码。

---

## 🤖 第三步：两种核心功能

Cursor 提供了两种主要的 AI 辅助功能，满足不同的编程需求：

---

## 🚀 方法一：Agent 智能体（`Ctrl + I`）- 复杂操作

**Agent（也叫 Composer）** 是 Cursor 最强大的功能，可以帮你从零开始编写完整的程序，或者对整个项目进行复杂的修改。

### 🎯 什么是 Agent？

Agent 就像一个会编程的助手，你只需要用**自然语言**（就像平时说话一样）告诉它你想要什么，它就会帮你写出代码。它可以同时创建和修改多个文件，理解整个项目的上下文。

### 🔧 如何使用 Agent？

1. **按 `Ctrl + I`**（Windows/Linux）或 **`Cmd + I`**（Mac）
   - 或者点击窗口右上角的 **Cursor 图标按钮**（在齿轮的左侧）
2. 屏幕下方会出现一个输入框
3. **用中文或英文描述你想要的功能**

> 💡 **提示**：这两种方法打开的是同一个 Agent 功能，你可以选择任意一种方式使用。

### 📝 示例：让 AI 帮你写一个计算器

**步骤：**

1. 创建一个新文件 `calculator.py`
2. 按 `Ctrl + I`（或 `Cmd + I`）
3. 在输入框中输入：

```
帮我创建一个简单的计算器程序，可以计算两个数字的加减乘除
```

4. 按 `Enter` 确认
5. AI 会自动生成代码！

**AI 可能会生成类似这样的代码：**

```python
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

def multiply(a, b):
    return a * b

def divide(a, b):
    if b == 0:
        return "错误：不能除以零"
    return a / b

# 使用示例
print(add(10, 5))      # 输出：15
print(subtract(10, 5)) # 输出：5
print(multiply(10, 5)) # 输出：50
print(divide(10, 5))   # 输出：2.0
```

### ✨ Agent 的强大之处

- ✅ **可以同时修改多个文件**：AI 会自动创建需要的所有文件
- ✅ **理解上下文**：AI 会查看你现有的代码，保持风格一致
- ✅ **自动修复错误**：如果代码有错误，告诉 AI，它会帮你修复
- ✅ **逐步完善**：你可以不断提出新需求，AI 会在现有代码基础上改进

### 💬 如何与 Agent 对话？

在 Agent 模式下，你可以：

1. **提出新需求**：

   ```
   添加一个功能：可以保存待办事项到文件
   ```

2. **修改现有功能**：

   ```
   把计算器的界面改得更漂亮一些
   ```

3. **修复问题**：

   ```
   程序报错了，帮我修复一下
   ```

   （记得把错误信息也告诉 AI，或者直接把终端输出添加到对话中）

4. **添加说明**：

   ```
   为代码添加中文注释，解释每一部分的作用
   ```

## ⚡ 方法二：快速编辑（`Ctrl + K`）- 局部修改

**`Ctrl + K`** 用于对选中的代码进行快速、局部的修改。适合修改一小段代码，而不是整个项目。

### 🎯 什么时候用 `Ctrl + K`？

- ✅ 修改一小段代码
- ✅ 改进代码写法
- ✅ 添加注释
- ✅ 修复局部错误
- ✅ 重构单个函数

### 🔧 如何使用 `Ctrl + K`？

1. **选中你想要修改的代码**
2. **按 `Ctrl + K`**（Windows/Linux）或 **`Cmd + K`**（Mac）
3. 输入你想让 AI 做什么
4. 按 `Enter` 确认

### 📝 使用示例

**示例 1：改进代码写法**

- 选中代码 → `Ctrl + K` → 输入："把这段代码改成更简洁的写法"

**示例 2：添加注释**

- 选中代码 → `Ctrl + K` → 输入："为这段代码添加中文注释"

**示例 3：修复错误**

- 选中出错的代码 → `Ctrl + K` → 输入："修复这个错误"

**示例 4：优化性能**

- 选中代码 → `Ctrl + K` → 输入："优化这段代码，让它运行得更快"

### 💡 Agent vs `Ctrl + K`：如何选择？

| 场景               | 使用哪个？              |
| ------------------ | ----------------------- |
| 创建新项目或新功能 | **Agent（`Ctrl + I`）** |
| 修改多个文件       | **Agent（`Ctrl + I`）** |
| 修改一小段代码     | **`Ctrl + K`**          |
| 改进单个函数       | **`Ctrl + K`**          |
| 添加注释或文档     | **`Ctrl + K`**          |
| 修复局部错误       | **`Ctrl + K`**          |

写完代码后，你需要运行它才能看到效果！

### Python 代码

### 如果遇到错误？

**不要慌！** 把错误信息告诉 AI：

1. 复制终端中的错误信息
2. 按 `Ctrl + I` 打开 Agent
3. 粘贴错误信息，说："程序报错了，帮我修复"
4. AI 会帮你找出问题并修复

## ❓ 常见问题

### Q: AI 生成的代码我看不懂怎么办？

**A:** 没关系！你可以：

1. 选中代码，按 `Ctrl + K`，输入："用简单的话解释这段代码的作用"
2. 或者按 `Ctrl + I`，问："这段代码是什么意思？用简单的话解释"

### Q: 代码运行出错了怎么办？

**A:**

1. 复制错误信息
2. 按 `Ctrl + I`，粘贴错误信息
3. 说："帮我修复这个错误"
4. AI 会帮你解决

### Q: 我想修改代码，但不知道怎么说？

**A:** 用最简单的话描述就行：

- "把按钮改成红色"
- "添加一个保存功能"
- "让程序运行得更快"
- "改成中文界面"

### Q: AI 生成的代码不是我想要的怎么办？

**A:**

1. 告诉 AI 哪里不对："这个功能不对，我想要的是..."
2. 或者更详细地描述你的需求
3. AI 会重新生成

## 🎯 学习建议

### ✅ 这样做：

1. **从简单开始**：先做简单的程序，比如计算器、待办列表
2. **多尝试**：不要怕出错，多试试不同的需求
3. **看代码**：即使看不懂，也看看 AI 生成的代码，慢慢就懂了
4. **问问题**：不懂就问 AI，它会耐心解释

### ❌ 避免这样：

1. **不要一开始就做太复杂的项目**：先熟悉基本操作
2. **不要完全依赖 AI**：尝试理解代码，这样你才能进步
3. **不要害怕修改**：AI 生成的代码可以随时修改

## 🏁 总结

**记住两种核心方法：**

1. **`Ctrl + I`（Agent）**：用于创建新项目、添加新功能、修改多个文件等复杂操作
2. **`Ctrl + K`**：用于修改选中的一小段代码，进行局部改进

**使用流程：**

1. **创建文件** → 按 `Ctrl + I`，告诉 Cursor 你想要什么
2. **运行代码** → 看看效果
3. **局部改进** → 选中代码，按 `Ctrl + K`，提出改进建议
4. **继续完善** → 按 `Ctrl + I`，添加新功能或修复问题

**最重要的是：**

- 用**自然语言**描述你的需求，就像和朋友聊天一样
- 不要担心说错，AI 会理解你的意思
- 多尝试，多练习，慢慢你就会了！
